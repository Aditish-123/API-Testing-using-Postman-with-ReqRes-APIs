# API-Testing-using-Postman-with-ReqRes-APIs
# 🔐 API Testing, Bug Reporting & SQL Validation using ReqRes API

## 📌 Project Overview
This project demonstrates API Testing using Postman with ReqRes APIs.  
The project includes CRUD operation testing, negative testing, bug reporting, and SQL validation concepts used in real-world QA workflows.

## 🚀 Features Covered
- GET API Testing
- POST API Testing
- PUT API Testing
- DELETE API Testing
- Status Code Validation
- Negative Testing
- Bug Reporting
- SQL Validation Concepts
- Collection Runner Execution
